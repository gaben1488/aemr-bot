# ПЛАН ДОЗАПОЛНЕНИЯ ВИКИ — сведение пробелов семи ролей

Источник пробелов: семь ролевых аудитов в `_kb/_audit/*.md`. Факты выверены по
коду (`bot/aemr_bot`) и `_kb/_LEDGER.md`. Дубли сведены: общий пробел получает
ОДИН канонический раздел-дом, в остальных — только перекрёстная ссылка.
Ложные пробелы (контент есть, но роль «не нашла») переквалифицированы в задачи
НАВИГАЦИИ.

Сборка вики: фрагменты `_kb2/<раздел>.html` → `docs/site/index.html`.
Разделы: zhitel, operator, admin, developer, security, pravovoe, koncept
(обзорный лендинг — obzor), spravochnik (справочник/глоссарий), adr (решения).

Легенда приоритетов: **P0** — реально мешает роли (ошибка/противоречие/важный
сценарий молчит); **P1** — полезно; **P2** — мелочь.
