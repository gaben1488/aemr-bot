# Банк болей роли «Житель» — аудит покрытия вики

Артефакт «боли роли». Призма: обычный житель Елизовского МО в мессенджере MAX,
без техзнаний («чайник»). Каждый вопрос/страх/непонимание за весь жизненный цикл
взаимодействия с ботом.

- **Источник вопросов:** мысленный проход роли + код `bot/aemr_bot` (воронка,
  меню, согласие, рассылка, гео, лимиты, блокировка, bot_stopped, export).
- **Проверка покрытия:** `docs/site/_kb2/zhitel.html` (+ смежные `pravovoe.html`,
  `spravochnik.html` при нужде).
- **Легенда:** ✓ отвечен прямо · ~ частично/вскользь/надо доискаться · ✗ пробел.

Дата: 2026-06-01.
