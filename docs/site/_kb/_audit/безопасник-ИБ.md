# Банк болей роли «Безопасник-ИБ» — аудит покрытия вики

Артефакт «боли роли». Призма: **ИБ-ответственный / аудитор информационной
безопасности** учреждения. Не пишет код, но отвечает за то, что система не утечёт
и пройдёт проверку. Думает категориями: модель угроз, точки ввода, удаление ПДн,
сроки хранения (retention), бэкап и его шифрование, антифишинг, реакция на
инцидент, компрометация секретов, аудит-журнал как доказательная база,
соответствие УЗ-4 и 152-ФЗ. Чайник в части именно этого бота, эксперт в части
«как вообще оценивают защищённость ИСПДн».

- **Источник вопросов:** мысленный проход всего жизненного цикла ИБ-сопровождения
  (приёмка → эксплуатация → инцидент → проверка регулятора → вывод из эксплуатации)
  + код `bot/aemr_bot` (db_backup, threat_intel, settings_store, admin_events,
  users, cron, config, _auth) + выверенные факты `_LEDGER.md`.
- **Проверка покрытия:** `docs/site/_kb2/security.html` (основной), смежные
  `pravovoe.html` (Политика ПДн, согласие, правовые основания), `admin.html`
  (сетевая изоляция, секреты на сервере), `operator.html` (антифишинг на смене),
  `spravochnik.html` (реестр переменных) — при нужде.
- **Легенда:** ✓ отвечен прямо · ~ частично / вскользь / в смежном разделе /
  надо доискаться · ✗ пробел.

Дата: 2026-06-01.
