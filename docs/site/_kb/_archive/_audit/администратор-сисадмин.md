# Банк болей роли «Администратор-сисадмин» — аудит покрытия вики

Артефакт «боли роли». Призма: человек, который **ставит и держит сервер** —
установка с нуля, `.env`, порты/health, бэкап и **восстановление**, обновление/
откат, диагностика сбоев, ротация секретов, отказы («упал / не стартует / диск
полон / сменился токен»), мониторинг, перенос на другой сервер. Чайник в части
именно этого бота, но с базовым Linux/Docker.

- **Источник вопросов:** мысленный проход всего жизненного цикла эксплуатации +
  код `bot/aemr_bot` (config, cron, health, db_backup, docker-compose, скрипты
  auto-deploy/healthwatch/audit_vps) + выверенные факты `_LEDGER.md`.
- **Проверка покрытия:** `docs/site/_kb2/admin.html` (основной), смежные
  `spravochnik.html` (реестр переменных, схема БД, шпаргалка консоли, мониторинг
  Postgres), `security.html`, `pravovoe.html`, `developer.html` — при нужде.
- **Легенда:** ✓ отвечен прямо · ~ частично/вскользь/надо доискаться/в смежном
  разделе · ✗ пробел.
