# Банк болей роли «Руководитель-аналитик» — аудит покрытия вики

Артефакт «боли роли». Призма: **руководитель отдела / новый член команды**, который
принимает бот в управление. Его вопросы — не «как нажать кнопку», а «зачем это
вообще, в чём ценность, чего бот сознательно НЕ делает и почему, как мерить
эффективность, какие риски, что будет при росте, куда развивать». Не айтишник, но
отвечает за результат канала перед Главой и жителями.

- **Источник вопросов:** мысленный проход всего жизненного цикла владения каналом
  (решение о запуске → пилот → рутина → разбор инцидента → отчёт наверх → решение
  о развитии/сворачивании) + продуктовые материалы проекта + выверенные факты
  `_LEDGER.md` (включая спецификацию аналитической панели и решения по ретеншену).
- **Проверка покрытия:** `docs/site/_kb2/koncept.html` (основной), смежные при нужде:
  `obzor.html` (управленческий обзор), `pravovoe.html` (сроки, SLA, ответственность),
  `operator.html` (статистика, SLA в работе), `admin.html` (эксплуатация, отказы),
  `security.html` (модель угроз).
- **Легенда:** ✓ отвечен прямо в koncept · ~ частично / вскользь / только в смежном
  разделе / надо доискаться · ✗ пробел (нет нигде по существу для этой призмы).

Дата: 2026-06-02. Источник истины — код (`bot/aemr_bot`), без галлюцинаций.
