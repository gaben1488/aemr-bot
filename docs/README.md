# Документация проекта «ЧАТ-БОТ / АДМИНИСТРАЦИЯ ЕМО»

Чат-бот обратной связи Администрации Елизовского муниципального округа в мессенджере MAX (`@aemo_chat_bot`, id 274913354).

## Главное — единая вики

Вся пользовательская и инженерная документация собрана в **самодостаточную офлайн-вики**:

- **[site/index.html](site/index.html)** — один HTML-файл, открывается двойным кликом, работает без интернета. Разделы: Обзор · Жителю · Оператору · Администратору · Разработчику · Как устроено · Безопасность и ПДн · Правовое · Справочник · Журнал решений. Поиск по всей базе (Ctrl/Cmd+K) и интерактивные диаграммы (на весь экран, скачивание SVG/PNG).
- Источник вики: `site/_kb2/*.html` (фрагменты разделов) + `site/_kb/_shell.html` (каркас, CSS, JS); собирается скриптом `site/_kb/_assemble.py` в `index.html`. Синхронность стережёт CI-тест `bot/tests/test_wiki_in_sync.py` и pre-commit-хук. **`index.html` руками не правят** — правят фрагменты и пересобирают.
